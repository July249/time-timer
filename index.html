How To Use It?

Setting

  • 시, 분, 초 영역을 클릭하면 커서가 깜빡이게 됩니다.
  • 시는 0부터 24까지, 분과 초는 0부터 59까지 입력이 가능합니다.
  • 값을 입력하지 않으면 00을 기본값으로 가집니다.

Start

  • Play 버튼을 클릭하면 타이머가 동작합니다.

Pause

  • 타이머가 동작 중일 때 마우스가 움직이면 정지 버튼이 나타납니다.
  • 정지 버튼을 클릭하면 타이머의 동작이 멈춥니다.
  • 타이머의 시, 분, 초를 클릭해도 타이머의 동작이 멈춥니다.
